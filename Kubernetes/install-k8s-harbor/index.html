1. 在安装好docker后安装Harbor私有镜像库

  • 默认用户名：admin
  • 默认密码：Harbor12345

1.1 Harbor依赖docker-compose，先安装docker-compose

1.2 重启docker后安装Harbor

  • docker登录Harbor
  • 如果报错:Error response from daemon: Get http: server gave HTTP response to HTTPS client
  • 修改配置文件(指定私有仓库，可忽略跳过)

1.3. 推送镜像到Harbor仓库

  • SOURCE_IMAGE[:TAG]：原镜像名字和tag
  • test：Harbor项目名称
  • IMAGE[:TAG]：自定义命名，以及版本标签
