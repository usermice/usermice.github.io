      • 10.1 ssh-keygen：用于创建密钥的程序
      • 10.2 -m PEM：将密钥的格式设为PEM
      • 10.3 -t rsa：创建密钥类型为：RSA格式，系统默认提供名为 id_rsa 的密钥对，有些工具可能要求私钥文件名为 id_rsa
      • 10.4 -b 4096：密钥的位数，本例中为4096
      • 10.5 -C "xxx@myserver"：追加到公钥文件末尾以便于识别的注释。 通常以电子邮件地址用作注释，但也可以使用任何最适合你基础结构的事物
      • 10.6 -f ~/.ssh/：私钥文件的文件名（如果选择不使用默认名称）。 追加了.pub的相应公钥文件在相同目录中生成。 该目录必须存在
      • 10.7 -N mypassphrase：用于访问私钥文件的其他密码
