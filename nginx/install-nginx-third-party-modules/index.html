install-nginx-third-party modules

发表于 2022-05-22 分类于 Nginx 阅读次数：
本文字数： 1k 阅读时长 ≈ 1 分钟

1. 安装Nginx GeoIP2IP定位模块

  • GeoIP2模块依赖libmaxminddb库

1.1 安装libmaxminddb库

$  wget -c https://github.com/maxmind/libmaxminddb/releases/download/1.3.2/libmaxminddb-1.3.2.tar.gz -P /opt/src/    ##下载到指定目录下:-c断点续传、-P下载指定目录，  
$ tar -zxvf /opt/src/libmaxminddb-1.3.2.tar.gz -C /opt/src/
$ cd /opt/src/libmaxminddb-1.3.2
$ ./configure
$ make -j2 && make install

1.2 安装GeoIP2

$ git clone https://github.com/leev/ngx_http_geoip2_module.git /opt/apps/nginx/ngx_http_geoip2_module
$ ./configure --prefix=/opt/apps/nginx --add-module=/opt/apps/nginx/ngx_http_geoip2_module

2. 安装ngx_brotli压缩算法

$ git clone https://github.com/google/ngx_brotli /opt/apps/nginx/ngx_brotli
$ cd /opt/apps/nginx/ngx_brotli
$ git submodule update --init  #更新ngx_brotli所需依赖库
$ ./configure --prefix=/opt/apps/nginx --add-module=/opt/apps/nginx/ngx_brotli

3. 安装ngx_http_upstream_check_module后端服务端口检测模块

$ git clone https://github.com/yaoweibin/nginx_upstream_check_module.git /opt/apps/nginx/nginx_upstream_check_module #下载检测包  
$ ./configure --prefix=/opt/apps/nginx --add-module=/opt/apps/nginx/nginx_upstream_check_module
