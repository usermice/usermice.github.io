1.1 部署NTP时间同步服务

    1. 下载安装包、解压、配置、安装
    1. 修改ntp.conf配置文件
  • 2.1 restrict default nomodify notrap noquery替换为restrict default nomodify允许任何客户机IP进行时间同步
  • 2.2 在restrict default nomodify notrap noquery默认拒绝所有之后增加一行：restrict 192.168.18.0 mask 255.255.255.0 nomodify
    1. 以守护进程启动ntpd
  • 3.1 ntpd启动后，客户机要等几分钟再与其进行时间同步，否则会提示no server suitable for synchronization found错误
    1. 在客户机器配置计划任务
  • 4.1 每天的5点13分、9点13分、14点13分、19点13分与时间同步服务器进行同步
