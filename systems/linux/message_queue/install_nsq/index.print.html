Install_Nsq

nsq部署

  • Nsq官网
  • 下载nsq预编译二进制安装包
$ wget -P /usr/local/src/ https://s3.amazonaws.com/bitly-downloads/nsq/nsq-1.2.1.linux-amd64.go1.16.6.tar.gz
  • 解压启动安装包
$ tar -zxvf nsq-1.2.1.linux-amd64.go1.16.6.tar.gz -C /usr/local/src/
$ cd nsq-1.2.1.linux-amd64.go1.16.6/bin/
$ nohup ./nsqlookupd &  	# 启动nsqlookup
$ nohup ./nsqd --lookupd-tcp-address=127.0.0.1:4160 &  	# nsqd
$ nohup ./nsqadmin --lookupd-http-address=0.0.0.0:4161 --http-address=IP:8761 & # nsqadmin管理，ip需要更换为服务所在的服务器IP
  • 测试、访问
$ curl -d 'hello world 1' 'http://127.0.0.1:4151/pub?topic=test'  	# 写入消息

$ ./nsq_to_file --topic=test --output-dir=/tmp/log --lookupd-http-address=127.0.0.1:4161  	# 把队列消息写入到文件
  • 通过浏览器访问nsqadmin
