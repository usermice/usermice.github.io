部署Tomcat服务

    1. Tomcat官网
    1. 8.3.x版本Tomcat安装包
    1. 安装Tomcat前准备
  • 1.1 创建Tomcat启动用户
$ groupadd myadmin && useradd -g myadmin myadmin  
$ echo "20210627@123"|passwd --stdin myadmin &>/dev/null

$ echo "#########Start/Stop Service###############
Cmnd_Alias MYADMIN_START_SERVICES = /etc/init.d/nginx, /etc/init.d/tomcat, /opt/* /usr/bin/kill
myadmin ALL=(ALL)NOPASSWD:MYADMIN_START_SERVICES" > /etc/sudoers.d/myadmin

$ chmod 660 /etc/sudoers.d/myadmin
  • 1.2 创建Tomcat部署目录
$ mkdir -p /opt/{data,apps,src,deploy,scripts,logs/tomcat}
  • 1.3 检测JDK环境
$ yum list |grep openjdk  	# 包有 @ 代表安装

$ echo "#######JDK環境變數########################  
export JAVA_HOME=/opt/data/jdk  
export JRE_HOME=$JAVA_HOME/jre  
export CLASSPATH=$JAVA_HOME/lib:$JAVA_HOME/lib/tools.jar  
export PATH=$PATH:$JAVA_HOME/bin:$JAVA_HOME/jre/bin    
CATALINA_HOME=/opt/data/tomcat  	# 部署多实例启动脚本应用    
export CATALINA_HOME ">>/home/myadmin/.bash_profile

$ source /home/myadmin/.bash_profile

$ ln -s /opt/data/jdk/bin/java /sbin/java  	# 使jar包可以直接运行,后面做软连接,授权
    1. 下载安装Tomcat
  • 2.1 下载Tomcat安装包
$ wget -P /opt/src/ https://archive.apache.org/dist/tomcat/tomcat-8/v8.5.35/bin/apache-tomcat-8.5.35.tar.gz 

$ wget -P /opt/src/ https://archive.apache.org/dist/tomcat/tomcat-8/v8.5.35/bin/apache-tomcat-8.5.35.tar.gz.sha512
  • 2.2 检测安装包完整性
# md5sum -c apache-tomcat-6.0.35.tgz.md5
$ sha512sum -c apache-tomcat-8.5.35.tar.gz.sha512
  • 2.3 解压以及更改权限
# ln -s /opt/lucky/apache-tomcat-8.5.35 /opt/data/tomcat  	# 可做软连接
$ tar -zxvf /opt/src/apache-tomcat-8.5.35.tar.gz -C /opt/data/
  • 2.4 修改Tomcat主配置文件
$ sed -i "s/8080/8000/g" /opt/data/tomcat/conf/server.xml 
$ sed -i 's/logs/\/opt\/data\/logs\/tomcat/g' /opt/data/tomcat/conf/server.xml
  • 2.5 修改Tomcat的Cataline.sh文件
$ sed -i '/# OS specific support/iJAVA_OPTS="-Dfile.encoding=UTF-8 -server -Xms2048m -Xmx2048m -Xmn1024m -XX:SurvivorRatio=10 -XX:MaxTenuringThreshold=15 -XX:NewRatio=2 -XX:+DisableExplicitGC"' /opt/data/tomcat/bin/catalina.sh

$ sed -i 's/"$CATALINA_BASE"\/logs\/catalina.out/\/opt\/data\/logs\/tomcat\/catalina.out/g' /opt/data/tomcat/bin/catalina.sh

# sed -i '/      <\/Host>/i\        <Context path="/" docBase="/opt/lucky/publish-luckybackend" reloadable="true"><\/Context>' /opt/data/tomcat/conf/server.xml

$ sed -i 's/${catalina.base}\/logs/\/opt\/data\/logs\/tomcat/g' /opt/data/tomcat/conf/logging.properties
  • 2.6 防火墙开放端口

  • 2.6.1 8005：端口是tomcat本身管理端口，修改一个不易猜到

  • 2.6.2 8080：端口是tomcat负责建立HTTP连接。在通过浏览器访问tomcat服务器的web应用时，使用的就是这个端口

  • 2.6.3 8009：tomcat负责和其他的HTTP服务器建立连接，如nginx和apache互通使用

$ firewall-cmd --permanent --new-ipset=tomcat_whitelist --type=hash:ip  
$ firewall-cmd --permanent --add-rich-rule="rule family='ipv4' source ipset='tomcat_whitelist' port prot='7000-7007' protocol='tcp' accept"  
$ firewall-cmd --reload
  • 2.7 网站根目录以及连接数据库
# 默认网站的主目录
$ ls /usr/local/tomcat/webapps/

$ cat >/usr/local/tomcat/webapps/ROOT/WEB-INF/config/jdbc.properties<<-EOF
# jdbc.url=jdbc:mysql://192.168.213.50:16303/jspgou?characterEncoding=UTF-8  	# jspgou代表数据库
# jdbc.username=root
# jdbc.password=123
EOF
  • 2.8 启动Tomcat
$ /opt/data/tomcat/bin/startup.sh
    1. 编写Tomcat启动文件
$ cat >tomcat.sh<<-EOF
#!/bin/bash
TOMCAT_PATH=/usr/local/tomcat

usage(){
   echo "Usage: $0 [start|stop|status|restart]"
}

status_tomcat(){
ps aux | grep java | grep tomcat | grep -v 'grep' 
}

start_tomcat(){
/usr/local/tomcat/bin/startup.sh
}

stop_tomcat(){
TPID=$(ps aux | grep java | grep tomcat | grep -v 'grep' | awk '{print $2}')
kill -9 $TPID
sleep 5;

TSTAT=$(ps aux | grep java | grep tomcat | grep -v 'grep' | awk '{print $2}')
    if [ -z $TSTAT ];then
      echo "tomcat stop"
    else
      kill -9 $TSTAT
    fi

cd $TOMCAT_PATH

rm temp/* -rf
rm work/* -rf
}

main(){
case $1 in
   start)
      start_tomcat;;
   stop)
      stop_tomcat;;
   status)
      status_tomcat;;
   restart)
      stop_tomcat && start_tomcat;;
    *)
      usage;
esac
}

main $1
EOF
