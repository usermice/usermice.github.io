安装配置JFrog Artifactory

    1. 下载最新JFrog Artifactory社区版镜像
$ docker pull docker.bintray.io/jfrog/artifactory-jcr:latest
    1. 为私有镜像仓库做持久化存储
$ mkdir -p /opt/docker/volume/artifactory
$ chmod 777 -R /opt/docker

# 切换到上述目录后执行数据卷创建命令
$ cd /opt/docker/volume/artifactory
$ docker volume create data
    1. 启动镜像
  • 3.1 默认用户名：admin
  • 3.2 默认密码：password
$ docker run --name jfrog-artifactory -d -v /opt/docker/volume/artifactory:/var/opt/jfrog/artifactory -p 8081:8081 -p 8082:8082 docker.bintray.io/jfrog/artifactory-jcr:latest
    1. 打开浏览器访问
http://ip:8082  	# 或者配置Nginx域名
