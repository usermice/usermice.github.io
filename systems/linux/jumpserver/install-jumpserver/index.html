安装Jumpserver堡垒机

    1. jump server简介、安装 JumpServer
  • 1.1 完全开源，使用GNU GPL v2.0开源协议，符合4A专业运维审计
  • 1.2 使用Python/Django进行开发，遵循Web2.0规范
  • 1.3 实现了跳板机基本功能，认证、授权、审计
  • 1.4 集成了Ansible批量命令，支持web Terminal
  • 1.5 采纳分布式架构，支持多机房跨区域部署
  • 1.6 现支持SSH、Telnet、RDP、VNC协议资产
  • 1.7 自动收集硬件信息、录像回放、命令搜索、实时监控、批量上传下载
  • 1.8 多租户：一套系统，多个子公司和部门同时使用
  • 1.9 多应用支持：数据库，Windows远程应用Kubernetes
  • 1.10 官方视频

基础配置环境要求

    1. 安装环境要求，默认账号密码都是：admin
系统/服务版本工具/数据库字符集
linux/amd64>= 4.0wget curl tar gettext iptables python
MySQL>= 5.7utf8、utf8_general_ci
MariaDB>= 10.2utf8mb3、utf8mb3_general_ci
Redis>= 5.0Sentinel模式，使用Redis做cache和celery broke
    1. 生成随机加密密钥
  • 2.1 Linux系统，生成随机加密密钥
$ if [ "$SECRET_KEY" = "" ]; then SECRET_KEY=`cat /dev/urandom | tr -dc A-Za-z0-9 | head -c 50`; echo "SECRET_KEY=$SECRET_KEY" >> ~/.bashrc; echo $SECRET_KEY; else echo $SECRET_KEY; fi

$ if [ "$BOOTSTRAP_TOKEN" = "" ]; then BOOTSTRAP_TOKEN=`cat /dev/urandom | tr -dc A-Za-z0-9 | head -c 16`; echo "BOOTSTRAP_TOKEN=$BOOTSTRAP_TOKEN" >> ~/.bashrc; echo $BOOTSTRAP_TOKEN; else echo $BOOTSTRAP_TOKEN; fi
  • 2.2 macOS系统，生成随机key
$ if [ "$SECRET_KEY" = "" ]; then SECRET_KEY=`LC_CTYPE=C tr -dc A-Za-z0-9 < /dev/urandom | head -c 50`; echo "SECRET_KEY=$SECRET_KEY" >> ~/.bash_profile; echo $SECRET_KEY; else echo $SECRET_KEY; fi  

$ if [ "$BOOTSTRAP_TOKEN" = "" ]; then BOOTSTRAP_TOKEN=`LC_CTYPE=C tr -dc A-Za-z0-9 < /dev/urandom | head -c 16`; echo "BOOTSTRAP_TOKEN=$BOOTSTRAP_TOKEN" >> ~/.bash_profile; echo $BOOTSTRAP_TOKEN; else echo $BOOTSTRAP_TOKEN; fi
    1. 创建数据库
$ DB_PASSWORD=$(cat /dev/urandom | tr -dc A-Za-z0-9 | head -c 24)  	# 生成随机数据库密码
$ echo -e "\033[31m 你的数据库密码是 $DB_PASSWORD \033[0m"
$ mysql -uroot -e "create database jumpserver default charset 'utf8'; grant all on jumpserver.* to 'jumpserver'@'127.0.0.1' identified by '$DB_PASSWORD'; flush privileges;"

# 创建数据库以及编码类型
mysql > create database jumpserver default charset 'utf8';

# 创建用户以及授权
mysql > grant all on jumpserver.* to 'jumpserver'@'%' identified by 'password';

Docker安装JumpServer

    1. 执行 docker 部署命令
  • 1.1 使用 docker-compose 标准部署
$ git clone --depth=1 https://github.com/jumpserver/Dockerfile.git /opt/docker-compose/jumpserver
  • 1.2 修改配置文件
$ cd /opt/docker-compose/jumpserver/
$ cp config_example.conf .env

$ cat >/opt/docker-compose/jumpserver/.env<<-EOF

Version=v2.28.7

## 构建参数, 支持 amd64/arm64
TARGETARCH=amd64

## Compose
COMPOSE_PROJECT_NAME=jms
## COMPOSE_HTTP_TIMEOUT=3600
## DOCKER_CLIENT_TIMEOUT=3600
DOCKER_SUBNET=172.16.238.0/24

## 持久化存储
VOLUME_DIR=/opt/jumpserver

## MySQL
DB_HOST=mysql
DB_PORT=3306
DB_USER=root
DB_PASSWORD=nu4x599Wq7u0Bn8EABh3J91G
DB_NAME=jumpserver

## Redis
REDIS_HOST=redis
REDIS_PORT=6379
REDIS_PASSWORD=8URXPL2x3HZMi7xoGTdk3Upj

## Core
SECRET_KEY=B3f2w8P2PfxIAS7s4URrD9YmSbtqX4vXdPUL217kL9XPUOWrmy
BOOTSTRAP_TOKEN=7Q11Vz6R2J6BLAdO
DEBUG=FALSE
LOG_LEVEL=ERROR

## Web
HTTP_PORT=80
SSH_PORT=2222
MAGNUS_PORT=30000-30020

##
## SECRET_KEY 保护签名数据的密匙, 首次安装请一定要修改并牢记, 后续升级和迁移不可更改, 否则将导致加密的数据不可解密。
## BOOTSTRAP_TOKEN 为组件认证使用的密钥, 仅组件注册时使用。组件指 koko、guacamole
EOF
  • 1.3 初始化数据库、启动jms_core容器
$ docker-compose -f docker-compose-network.yml -f docker-compose-init-db.yml up -d
  • 1.4 处理数据库合并
$ docker exec -i jms_core bash -c './jms upgrade_db'
  • 1.5 构建启动Jumpserver
$ docker-compose -f docker-compose-network.yml -f docker-compose.yml up -d

脚本安装Jumpserver

    1. 安装完成后配置文件/opt/jumpserver/config/config.txt Jumpserver 
$ curl -sSL https://github.com/jumpserver/jumpserver/releases/download/v2.24.2/quick_start.sh | bash
$ cd /opt/jumpserver-installer-v2.24.2

# 启动
$ ./jmsctl.sh start -d  	# 后台运行使用 -d 参数./jms start -d

# 停止
$ ./jmsctl.sh down

# 卸载
$ ./jmsctl.sh uninstall

# 帮助
$ ./jmsctl.sh -h
