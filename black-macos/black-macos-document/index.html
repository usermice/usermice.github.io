  • acdtinfo：检测当前机器的kexts安装情况

  • kpdescribe：调式Debug排错使用，用于恢复堆栈跟踪
  • LegacyBoot：在旧电脑中模拟UEFI环境的工具和脚本
  • LogoutHook：用于模拟NVRAM保存的增强脚本
  • macrecovery：启动盘制作脚本
  • macserial：序列号生成工具
  • ocpasswordgen：OpenCore密码数据生成工具
  • ocvalidate：检测config.list语法的正确与否

  • BOOT

BOOT引导文件夹，存放BOOTx64.ef引导文件

  • OC/ACPI(Advanced Configuration and Power Interface)：高级配置与电源接口

​ 存放这SSDT编译好的文件，格式均为.aml格式

  • Embedded controllers(EC)：嵌入式控制器
  • 所有半现代的Intel机器都有一个EC（通常称为H_EC、ECDV、EC0等）暴露在它们的DSDT中，许多AMD系统也暴露了它。这些控制器通常与macOS不兼容，可能会导致恐慌，因此需要对macOS隐藏。macOS Catalina需要一个名为的设备EC存在，因此创建了一个虚拟EC
  • 对于笔记本电脑，实际的嵌入式控制器仍然需要启用电池和热键才能工作，重命名EC还会导致Windows出现问题，因此最好在不禁用真正的嵌入式控制器的情况下创建假的EC
  • Plugin(插件)type(类型)

​ 这允许使用XCPM在**Intel** Haswell和更新的CPU上提供本机CPU电源管理，SSDT将连接到CPU的第一个线程。不适用于AMD

  • AWAC system clock(系统时钟)

​ 这适用于所有300系列主板，包括许多Z370主板，具体问题是较新的主板出厂时启用了AWAC时钟。这是一个问题，因为macOS无法与AWAC时钟通信，因此这要求我们要么强制使用传统的RTC时钟，要么在不可用的情况下创建一个假的供macOS使用

  • NVRAM(非易失存储器)SSDT(固态硬盘)

​ 真正的300系列主板（非Z370）不会在ACPI中将FW芯片声明为MMIO，因此内核会忽略UEFI内存映射声明的MMIO区域。这个SSDT带回了NVRAM支持

  • Backlight(背光)SSDT

​ 用于修复笔记本电脑上的背光控制支持

  • GPIO SSDT

​ 用于创建存根以允许VoodooI2C连接到，仅适用于笔记本电脑

  • XOSI SSDT(固态硬盘)

​ 用于将OSI调用重新路由到此SSDT，主要用于诱使我们的硬件认为它正在启动Windows，以便我们获得更好的触控板支持。这是一个以破坏Windows启动而闻名的非常hacky的解决方案，请改用GPIO SSDT

  • IRQ SSDT and(和)ACPI patches(补丁)

​ 需要修复DSDT中的IRQ冲突，主要用于笔记本电脑。SSDT时间独家

​ 注意Skylake和更新的系统很少有IRQ冲突，这主要在Broadwell和更早的系统上普遍存在

  • OC/Drivers：存放驱动文件，这些驱动均依 .efi后缀结尾
  • AudioDxe.efi：用与启动的时候播放Duang的声音，就像白苹果那样
  • CrScreenshotDxe.efi：OC引导界面截图驱动，按F10会保存当前界面的截图到EFI分区的根目录下
  • HiiDatabase.efi：用于支持UEFI字体渲染，四代酷睿后一般不需要
  • NvmExpressDxe.efi：用于让老主板支持NVME Express设备，四代酷睿后一般不需要
  • OpenCanopy.efi：使用图形化OC主题必备驱动
  • OpenHfsPlus.efi：文件系统驱动，用于支持识别HFS+的磁盘格式
  • OpenLinuxBoot.efi：OC 0.7.3新增的驱动，用于引导Linux系统
  • OpenPartitionDxe.efi：分区管理驱动程序。用于加载旧版macOS的DMG映像
  • OpenRuntime.efi：OC核心必备驱动，功能比较强大，大家记住这个是必备的驱动就行
  • OpenUsbKbDxe.efi：USB键盘驱动，用于模拟苹果热键，是KeySupport的等效方案
  • Ps2KeyboardDxe.efi：PS/2键盘驱动
  • Ps2MouseDxe.efi：PS/2鼠标驱动
  • UsbMouseDxe.efi：USB鼠标驱动，有些虚拟机需要依赖改驱动才可以在引导界面使用鼠标
  • XhciDxe.efi：XHCI USB controller驱动程序，基本上2代酷睿开始大多数固件都自带这个驱动程序了
