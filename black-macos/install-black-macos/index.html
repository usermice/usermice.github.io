BOOT引导文件夹，存放BOOTx64.ef引导文件

  • 1.2 OpenCore-0.9.1-RELEASE/X64/EFI/OC/ACPI(Advanced Configuration and Power Interface)：高级配置与电源接口

存放这SSDT编译好的文件，格式均为.aml格式

  • 1.2.1 Embedded controllers(EC)：嵌入式控制器

  • 所有半现代的Intel机器都有一个EC（通常称为H_EC、ECDV、EC0等）暴露在它们的DSDT中，许多AMD系统也暴露了它。这些控制器通常与macOS不兼容，可能会导致恐慌，因此需要对macOS隐藏。macOS Catalina需要一个名为的设备EC存在，因此创建了一个虚拟EC

  • 对于笔记本电脑，实际的嵌入式控制器仍然需要启用电池和热键才能工作，重命名EC还会导致Windows出现问题，因此最好在不禁用真正的嵌入式控制器的情况下创建假的EC

  • 1.2.2 Plugin(插件) type(类型)
  • 这允许使用XCPM在**Intel** Haswell和更新的CPU上提供本机CPU电源管理，SSDT将连接到CPU的第一个线程。不适用于AMD
  • 1.2.3 AWAC system clock(系统时钟)
  • 这适用于所有300系列主板，包括许多Z370主板，具体问题是较新的主板出厂时启用了AWAC时钟。这是一个问题，因为macOS无法与AWAC时钟通信，因此这要求我们要么强制使用传统的RTC时钟，要么在不可用的情况下创建一个假的供macOS使用
  • 1.2.4 NVRAM(非易失存储器) SSDT(固态硬盘)
  • 真正的300系列主板（非Z370）不会在ACPI中将FW芯片声明为MMIO，因此内核会忽略UEFI内存映射声明的MMIO区域。这个SSDT带回了NVRAM支持
  • 1.2.5 Backlight(背光) SSDT
  • 用于修复笔记本电脑上的背光控制支持
  • 1.2.6 GPIO SSDT
  • 用于创建存根以允许VoodooI2C连接到，仅适用于笔记本电脑
  • 1.2.7 XOSI SSDT(固态硬盘)
  • 用于将OSI调用重新路由到此SSDT，主要用于诱使我们的硬件认为它正在启动Windows，以便我们获得更好的触控板支持。这是一个以破坏Windows启动而闻名的非常 hacky 的解决方案，请改用GPIO SSDT
  • 1.2.8 IRQ SSDT and(和) ACPI patches(补丁)
  • 需要修复 DSDT 中的IRQ冲突，主要用于笔记本电脑。SSDT时间独家
  • 注意Skylake和更新的系统很少有IRQ冲突，这主要在Broadwell和更早的系统上普遍存在
  • 1.3 OpenCore-0.9.1-RELEASE/X64/EFI/OC/Drivers：存放驱动文件，这些驱动均依 .efi后缀结尾
  • AudioDxe.efi：用与启动的时候播放 Duang 的声音，就像白苹果那样
  • CrScreenshotDxe.efi：OC引导界面截图驱动，按F10会保存当前界面的截图到EFI分区的根目录下
  • HiiDatabase.efi：用于支持UEFI字体渲染，四代酷睿后一般不需要
  • NvmExpressDxe.efi：用于让老主板支持NVME Express设备，四代酷睿后一般不需要
  • OpenCanopy.efi：使用图形化OC主题必备驱动
  • OpenHfsPlus.efi：文件系统驱动，用于支持识别HFS+的磁盘格式
  • OpenLinuxBoot.efi：OC 0.7.3新增的驱动，用于引导Linux系统
  • OpenPartitionDxe.efi：分区管理驱动程序。用于加载旧版macOS的DMG映像
  • OpenRuntime.efi：OC核心必备驱动，功能比较强大，大家记住这个是必备的驱动就行
  • OpenUsbKbDxe.efi：USB键盘驱动，用于模拟苹果热键，是KeySupport的等效方案
  • Ps2KeyboardDxe.efi：PS/2键盘驱动
  • Ps2MouseDxe.efi：PS/2鼠标驱动
  • UsbMouseDxe.efi：USB鼠标驱动，有些虚拟机需要依赖改驱动才可以在引导界面使用鼠标
  • XhciDxe.efi：XHCI USB controller驱动程序，基本上2代酷睿开始大多数固件都自带这个驱动程序了
  • 1.4 OpenCore-0.9.1-RELEASE/X64/EFI/OC/ Kexts
  • 1.4.1 这里放着一些Kexts内核扩展文件，格式均为.kext文件
  • VirtualSMC.kext 模拟苹果系统的SMC芯片、只支持OS X 10.6+版本的系统
  • Lilu.kext Kexts的依赖，没有Liu就无法正常使用AppleALC、WhateverGreen、VirtualSMC等
  • VirualSMC插件

  • SMCProcessor.kext：用于监控Intel CPU温度、不适用基于AMD CPU系统、需要Mac OX X 10.7以上版本

  • SMCAMDProcessor 用于监控基于AMD Zen系统的CPU、需要MaOS 10.13以上版本

  • SMCRadeonGPU 用于监控AMD GPU系统上GPU的温度、需要MacOS 11以上版本

  • SMCSuperIO.kext用于监控风扇转速、不适用基于AMD CPU系统、需要MacOS X 10.6以上版本

  • SMCLightSensor.kext用于笔记本上的环境光传感器、不适用台式、需要MacOS X 10.6以上版本

  • SMCBatteryManager.kext用于测量笔记上电池读数、不使用台式、需要MacOS X 10.4以上版本

  • SMCDellSensors.kext用于检测戴尔计算机风扇、只支持戴尔系统

  • Graphics显卡驱动
  • WhateverGreen.kext 用于图形修补、DRM修复、主板ID检查、帧缓冲区修复等；所有GPU都受益于这个kext，请注意，仅笔记本电脑和AIO需要包含的SSDT-PNLF.dsl文件，请参阅 ACPI入门 了解更多信息，需要Mac OS X 10.6或更新版本
  • Audio声卡驱动
  • AppleALC.kext 用于AppleHDA修补，允许支持大多数板载声音控制器，AppleALCU.kext是AppleALC的精简版，仅支持数字音频 - 但您仍然可以在纯数字音频系统上使用AppleALC.kext，AMD 15h/16h可能存在AppleALC问题，Ryzen/Threadripper系统很少支持麦克风，需要OS X 10.4或更新版本
  • VoodooHDA.kext 比较古老且经典的声卡驱动，也叫万能声卡驱动，如果AppeALC.kext无法驱动话可以考虑这个，但是使用体验完美度肯定不如原生的AppleALC.kext，只支持OS X 10.6+版本的系统
  • Ethernet(以太网)有线网卡驱动
  • AtherosE2200Ethernet.kext Atheros高通和Killer 杀手网卡需要，注意：Atheros Killer E2500型号实际上是基于Realtek的，所以请使用RealtekRTL8111驱动
  • IntelMausi.kext 大多数Intel因特尔的网卡驱动，基于I211的芯片组的网卡需要使用SmallTreeIntel82576 kext，官方支持Intel的82578、82579、I217、I218和I219网卡，详细支持驱动的有线网卡型号可以参考 IntelMausi ，需要OS X 10.9或更新版本，10.6-10.8的老用户可以使用IntelSnowMausi替代
  • LucyRTL8125Ethernet.kext Realtek的2.5Gb的网卡驱动，官方这个页面需要注册才可以下载，也可以使用 蓝奏云 ，需要macOS 10.15+版本的系统
  • RealtekRTL8111.kext 大多数Realtek的千兆网卡驱动，注意：有时最新版本的kext可能无法正常工作，这个时候可以尝试使用旧版本
  • SmallTreeIntel82576.kext I211有线网卡驱动，大多数配备intel有线网卡的AMD主板需要，版本支持情况 OS X 10.9-12(v1.0.6)、macOS 10.13-14(v1.2.5)、macOS 10.15+(v1.3.0)
  • USB驱动
  • USBToolBox.kext
  • UTBMap.kext
  • XHCI-unsupported.kext 用于USB定制完成后USB 3.X接口依然无法正常工作，常用于400系列及其以上的主板，主板不是macOS原生支持的USB驱动的话需要这个
  • WiFi无线网卡驱动
  • Intel无线网卡系列
  • AirportItlwm.kext Intel网卡的WiFi驱动，支持驱动的 Intel无线网卡型号表 ，只支持macOS 10.13以及更高的版本
  • IntelBluetoothFirmware.kext与IntelBluetoothFirmware.kext Intel网卡的蓝牙驱动，与AirportItlwm.kext搭配使用，只支持macOS 10.13以及更高的版本，如果确定你的网卡型号支持驱动，如果确定你的网卡型号支持驱动，那么多半是USB没有定制好，macOS 12后蓝牙驱动的姿势又有变化
