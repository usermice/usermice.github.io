  • 安装hugo博客系统

dream

把博客推送到远端

  • 现在GitHub创建一个新项目，项目名称必须是以用户名，开头以：github.io结尾，
  • hugo配置使用主题以及GitHub地址
  • –theme： 指定使用主题
  • –baseUrl： 指定远端地址
  • –buildDrafts： 启动hugo
  • 在项目目录下生成public 目录
  • 把public目录推送到远端
