    1. 配置Zabbix-in-Telegram
  • 6.1 Zabbix公共存储库 

$ wget https://git.zabbix.com/rest/api/latest/projects/ZBX/repos/zabbix/archive?at=refs%2Fheads%2Frelease%2F4.4&format=zip
  • 6.2 下载zabbix-Telegram插件
$ yum -y install python-pip
$ git clone https://github.com/ableev/Zabbix-in-Telegram.git
$ cd /opt/zabbix/Zabbix-in-Telegram
$ pip install -r requirements.txt
  • 6.3 修改zabbix_server.conf配置文件

$ sed -i "s/# AlertScriptsPath/AlertScriptsPath/" /opt/zabbix/etc/zabbix_server.conf
  • 6.4 复制告警脚本到Zabbix Server
$ cp zbxtg.py zbxtg_settings.example.py zbxtg_group.py /usr/lib/zabbix/alertscripts/ $ cd /usr/lib/zabbix/alertscripts/ 
$ mv zbxtg_settings.example.py zbxtg_settings.py
  • 6.5 修改zbxtg_settings.py脚本
$ cat /usr/lib/zabbix/alertscripts/zbxtg_settings.py
tg_key = "XXX:YYY"  	# telegram bot api key, Telegram 机器人的ID和密钥 
zbx_server = "http://youip:port/zabbix/"  	# zabbix server full url zabbix 服务器地址
zbx_api_user = "Admin"  	# zabbix 用户名
zbx_api_pass = "zabbix"  	# zabbix 密码
  • 6.6 Zabbix-Web→报警媒介类型→脚本参数，填写
{ALERT.SENDTO}
{ALERT.SUBJECT}
{ALERT.MESSAGE}
--group

配置Linux系统登录告警

    1. 配置sshd登录告警
$ env
$ cat >/etc/ssh/sshrc<<-EOF
#!/bin/sh
a_user=$USER
a_ip=`echo $SSH_CLIENT | awk '{print $1}'`
a_time=$(date +"%F %H:%M:%S")
a_server=`/sbin/ifconfig eth0|sed -n '2p' |awk '{print $2}'`
msg="Someone logged in the Machine, please verify the IP
Time: ${a_time}
User: ${a_user}
Host: ${a_server}
Login ip: ${a_ip}"

###########dash unsupport array#################
TrustIp='192.168.0.61 192.168.0.37'
result=$(echo ${TrustIp} | grep "${a_ip}")

if [ "${result}" != "" ]; then
   exit
else
   curl -X POST "https://api.telegram.org/botID:KEY/sendMessage" -d "chat_id=GrouID&text=${msg}" > /dev/null
fi
EOF
